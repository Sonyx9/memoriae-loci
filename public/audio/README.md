# Zvukové stopy článků

Sem vložte zvukové soubory (m4a, mp3), na které odkazují články v `audioUrl`.

## Článek: Politický proces s Libereckými skauty (1952)

1. Stáhněte soubor z Google Drive:  
   [Případ_Tomsa_a_likvidace_libereckých_skautů.m4a](https://drive.google.com/file/d/1p-zqeinUdJCKE5ignBA8V-CxyvR43j5i/view?usp=sharing)
2. Přejmenujte na: `pripad-tomsa-libereckych-skautu.m4a`
3. Zkopírujte sem: `public/audio/pripad-tomsa-libereckych-skautu.m4a`

Cesta na webu bude: `/audio/pripad-tomsa-libereckych-skautu.m4a`
